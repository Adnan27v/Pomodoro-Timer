# Pomodoro Timer

A simple and elegant Pomodoro Timer application built with Python and Tkinter. This application helps you manage your work and break times effectively using the Pomodoro Technique.

## Features

- 25-minute work sessions
- 5-minute short breaks
- 20-minute long breaks after 4 work sessions
- Visual timer display
- Start and Reset functionality
- Checkmark tracking for completed work sessions
- Clean and intuitive user interface

## Requirements

- Python 3.x
- Tkinter (usually comes with Python installation)

## Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/pomodoro-timer.git
cd pomodoro-timer
```

2. Make sure you have Python installed on your system.

3. Run the application:
```bash
python main.py
```

## Usage

1. Click the "Start" button to begin your Pomodoro session
2. Work for 25 minutes
3. Take a 5-minute break
4. Repeat the cycle
5. After 4 work sessions, take a longer 20-minute break
6. Use the "Reset" button to reset the timer at any time

## Project Structure

- `main.py` - Main application file containing the Pomodoro timer logic and UI
- `tomato.png` - Timer image used in the application
